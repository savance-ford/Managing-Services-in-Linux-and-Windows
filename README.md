# Managing-Services-in-Linux-and-Windows
Managing Services in Linux and Windows

## Learning Objectives

- Getting familiar with common system services on Linux

- Starting, stopping, reloading services on Linux

- Editing service configuration

## Introduction

As a system administrator, you’ll need to know how to look at the status of a running service, and how to stop, start, and restart running services. You’ll also need to know how to configure services and fix problems you may encounter while running them. In this lab, we’ll focus on how to do this actions on Linux and Windows
